# Dev. 角色制作队列

> 由于制作角色的流程极度复杂（见其他开发文档），且当前开发组制作速度不足以支撑当前的角色制作，所以，角色制作采用排队模式，请各位成员原谅，谢谢！
>
> 本文档由Skyc8266管理，并依据文档分配角色制作工作任务

## 队列规则

- 角色制作队列分为三个优先级，第一优先级每人仅可上报一个，第二优先级每人至多2个，第三优先级每人至多3个
- 角色制作顺序按照优先级队列分配，每轮先进行第一优先级，随后是第二优先级，两个队列完成后进入下一轮填报与制作
- 第三优先级队列始终以缓慢速度进行制作，且总角色组数不超过15组，先到先得
- 第一优先级队列总角色组数不超过6组，先到先得
- 第二优先级队列总角色组数不超过10组，先到先得
- 如果提交制作该角色的人自己都没搞明白其人设与世界观（或者角色本身人设和世界观就极度模糊的），那么，恭喜你，这个角色不管你提交到哪个队列，都会被直接下降到第三队列，严重的不予制作。
- 如果你的角色很恶心人，那么，在组内投票后，决定是否制作（严重的直接不予制作，无需投票）
- **若提交制作涉及违法的内容，则直接清除有关数据。首次警告加禁止提交资产制作至少275天，第二次再犯，直接从隙间开发组永久移除提交者。十分严重的，直接移出开发者、并将证据等信息上报公安**

## 角色制作速度

- 对于世界观复杂的角色（例如鸣潮中的今汐、陆赫斯等），从零开始制作预计25～32天
- 对于世界观较为简单的角色（例如千恋万花中的芳乃），从零开始制作，预计5～14天
- 通常为4组角色同时制作
