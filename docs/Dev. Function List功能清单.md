# Dev. Function List功能清单

> 本文档由Skyc8266创建，仅供开发使用！

## A. 用户功能

### A1. 记忆系统

- 针对不同角色分别管理记忆，使用SQLITE数据库存储

#### A1.1 用户管理与自动备份

- **自动备份受保护模块管理**
- 用户可以手动管理每个角色的记忆条目

#### A1.2 系统自动记忆管理与强制对话调用

- 每个角色都有独立的记忆配置（例如**记忆力**与其他自定义配置），依据这些配置，系统会自动决定其长期记忆量（可变）、执行不同的记忆条目调用需要多长时间与短期记忆的遗忘速度等
- Ai在回复中涉及到任何过往信息的，必须从记忆库中调取，多数长期记忆与遗忘程度极低的短期记忆将会在每次开启新对话时自动载入到上下文中，最大载入量受角色记忆配置影响
